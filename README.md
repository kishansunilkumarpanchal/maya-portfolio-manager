# Maya Portfolio Manager

This repository is being organized as a monorepo with separate `backend` and `frontend` applications.

## Structure

- `backend/` - planned Python backend service area for future FastAPI work
- `frontend/` - planned React frontend application area
- `reference/legacy_flask/portfolio_v2/` - legacy Flask reference application kept intact

## Current Status

This scaffold intentionally contains no business logic yet. It is only the initial project structure and documentation layer for the next phase of development.

## Notes

- The legacy Flask code remains untouched.
- New development should target the new `backend` and `frontend` folders.
